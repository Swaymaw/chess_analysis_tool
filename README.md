## Chess Analysis Tool - MCP

- This tool uses an MCP backend to generate useful insights for a chess game using a PGN file. The MCP backend can use a variety of different chess engines to use for analysis.
- The goal is to create an AI Coach that can help evaluate chess games and help players improve their games, focus on what matter the most more effectively.

Features:

1. Game Summary features that gives a broad insight on each move played and it's impact on the game.
2. Per-move in detailed analysis for better reporting and move understanding.
